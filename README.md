# xenforo-spanish-translation-2.3.7

Traducción al español (ES) para XenForo versión 2.3.7.

## Cómo importarlo en el Panel de Administración
- Inicia sesión en el `Admin CP` de XenForo (acceso en `/admin.php`, por ejemplo `https://tu-dominio/admin.php`).
- Ve a `Apariencia` (`Appearance`) > `Idiomas` (`Languages`).
- Pulsa `Importar` (`Import`).
- Selecciona el archivo `language-Espanol-(ES).xml` de este repositorio.
- Elige si deseas crear un nuevo idioma o actualizar uno existente y confirma.
- Espera a que finalice el proceso de importación.

## Traducciones de addons
- Group membership moderators — versión 2.3.0
- Lazy Image Loader — versión 2.8.0
- Moderator Essentials — versión 2.6.1
- Standard Library — versión 1.22.6
- User Essentials — versión 4.5.9
- [BS] Clone User Group — versión 1.0.1

## English README

Spanish (ES) translation for XenForo version 2.3.7.

## How to import in the Admin Control Panel
- Log in to XenForo `Admin CP` (access at `/admin.php`, e.g. `https://your-domain/admin.php`).
- Go to `Appearance` > `Languages`.
- Click `Import`.
- Select the `language-Espanol-(ES).xml` file from this repository.
- Choose whether to create a new language or update an existing one and confirm.
- Wait for the import process to complete.
