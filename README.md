# xenforo-spanish-translation-2.3.7

Traducción al español (ES) para XenForo versión 2.3.7.

## Cómo importarlo en el Panel de Administración
- Inicia sesión en el `Admin CP` de XenForo (acceso en `/admin.php`, por ejemplo `https://tu-dominio/admin.php`).
- Ve a `Apariencia` (`Appearance`) > `Idiomas` (`Languages`).
- Pulsa `Importar` (`Import`).
- Selecciona el archivo `language-Espanol-(ES).xml` de este repositorio.
- Elige si deseas crear un nuevo idioma o actualizar uno existente y confirma.
- Espera a que finalice el proceso de importación.
